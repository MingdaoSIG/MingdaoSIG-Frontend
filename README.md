## 關於平台

明道 SIG 是專為明道中學師生打造的社群平台，讓每位成員都能輕鬆分享學習心得、課程筆記與活動紀錄。透過 Markdown 編輯器撰寫文章，你的想法可以用豐富的格式呈現給所有人。

## 功能特色

- **撰寫與發佈文章** — 使用內建的 Markdown 編輯器，支援即時預覽，撰寫過程自動儲存，不怕內容遺失
- **瀏覽與探索** — 首頁以無限捲動方式呈現最新文章，隨時掌握社群動態
- **公告系統** — 重要消息會置頂顯示 7 天，確保不錯過任何重要資訊
- **個人主頁** — 每位使用者都有專屬頁面，展示個人發表的所有文章
- **數據統計** — 以圖表方式呈現平台使用數據與趨勢
- **管理後台** — 社團幹部與指導老師可管理成員、文章與社團設定

## 如何使用

1. 前往 **[sig.mingdao.edu.tw](https://sig.mingdao.edu.tw)**
2. 使用明道校園 Google 帳號（`@ms.mingdao.edu.tw`）登入
3. 開始瀏覽文章，或點擊「新增文章」分享你的想法

## 常見問題

#### 誰可以使用這個平台？

所有擁有明道中學校園 Google 帳號（`@ms.mingdao.edu.tw`）的師生都可以登入使用。

#### 文章支援哪些格式？

平台使用 Markdown 語法，支援標題、粗體、斜體、程式碼區塊、圖片、連結等豐富格式。編輯時右側會即時顯示預覽。

#### 撰寫到一半的文章會消失嗎？

不會。編輯器會自動將內容儲存到瀏覽器中，即使不小心關閉頁面，下次開啟時仍可繼續編輯。
